SVGGraphics2D
=============

Ubuntu / Wheezy quick start:
```bash
$ sudo apt-get install git maven openjdk-7-jdk
$ git clone https://github.com/AndyHowlettGitHub/SVGGraphics2D.git
$ cd SVGGraphics2D
$ mvn clean test
```
Look in the directory target/test-classes/uk/ac/cam/ch/ucc/svggraphics2d/ - the SVG file in there should be of a triazole (view in e.g. Firefox).
